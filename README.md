# Codigos Reaproveitaveis SQL, jQuery e Javascript #

* Arquivo contendo diversas funcoes em jquery, genericas e reaproveitaveis para desenvolvimento de novos projetos no Orquestra BPM V3.

### util_js_v3.js ###
